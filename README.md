# AI+校招每日资讯简报

> 每日精选 AI 行业动态 + 校招资讯，一页掌握当天最值得关注的科技与招聘信息。

## 查看最新简报

👉 [点击查看最新简报](index.html)

支持按日期切换，所有历史简报均可查阅。

## 内容板块

| 板块 | 说明 |
|------|------|
| 今日头条 | 3 条重磅新闻，附来源链接和通俗解读 |
| 行业热点速览 | 5-6 条动态表格 |
| 办公软件 AI 动态 | WPS 灵犀、钉钉、飞书等 |
| 校园招聘资讯 | 当天宣讲会、招聘信息 |
| 每日 AI 新工具 | 值得关注的新工具推荐 |
